Nazariy asoslar

Maple dasturlash muhitida shartli va tsiklik tuzilmalar matematik hisoblashlarni avtomatlashtirish uchun ishlatiladi. Bu tuzilmalar orqali murakkab algoritmlarni oddiy va tushunarli ko'rinishda yozish mumkin.

Shartli tuzilma (if-then-else)

if shart then
    # bajarilishi kerak bo'lgan amallar
elif boshqa_shart then
    # boshqa amallar
else
    # aks holda bajarilishi kerak
end if;

For tsikli

for i from boshlang'ich to oxirgi by qadam do
    # takrorlanadigan amallar
end do;

While tsikli

while shart do
    # shart to'g'ri bo'lsa takrorlanadi
end do;

  • For va while tsikllarini ishlatish.

  • Algoritmik fikrlash ko'nikmalarini rivojlantirish.

Misollar & Maple kod namunalari

1) Juft/toq sonlarni aniqlash

Shartli tuzilma orqali sonlarni tasniflash:

# Sonni tekshirish
n := 15;

if n mod 2 = 0 then
    print(n, "juft son");
else
    print(n, "toq son");
end if;

# Natija: 15 toq son

2) Sonlar qatorini hisoblash

For tsikli orqali yig'indi hisoblash:

# 1 dan 100 gacha yig'indi
summa := 0;

for i from 1 to 100 do
    summa := summa + i;
end do;

print("Yig'indi:", summa);
# Natija: 5050

3) Funktsiya grafigi

Shartli funktsiya va uning grafigi:

# Qismli aniqlangan funktsiya
f := x -> if x >= 0 then 
    x^2 
else 
    -x^2 
end if;

plot(f(x), x=-5..5, 
    color=blue);

4) Faktorial hisoblash

While tsikli orqali faktorial:

# n! hisoblash
n := 5;
faktorial := 1;
i := 1;

while i <= n do
    faktorial := faktorial * i;
    i := i + 1;
end do;

print(n, "! =", faktorial);

5) Ro'yxat bilan ishlash

Ro'yxat elementlarini filterlash:

# Musbat sonlarni ajratish
sonlar := [-3, 7, -1, 4, 0, 9];
musbat := [];

for son in sonlar do
    if son > 0 then
        musbat := [op(musbat), son];
    end if;
end do;

print(musbat);
# Natija: [7, 4, 9]

6) Animatsiyali grafik

Vaqtga bog'liq funktsiya animatsiyasi:

# Sinus to'lqini animatsiyasi
with(plots):

frames := [];
for t from 0 to 20 do
    frame := plot(sin(x-t/5), 
        x=0..10, y=-2..2);
    frames := [op(frames), frame];
end do;

display(frames, insequence=true);

Masalalar (mustaqil ish)

Boshlang'ich daraja

  1. 1 dan 20 gacha bo'lgan barcha toq sonlarni chiqaradigan for tsiklini yozing.
  2. Berilgan son tub son yoki yo'qligini aniqlovchi if-else tuzilmasini tuzing.
  3. Kvadrat funktsiya $y = x^2$ ni $x \in [-5, 5]$ oraliqda chizing.

O'rta daraja

  1. Fibonacci sonlarini hisoblovchi while tsikli yozing (10 ta son).
  2. Qismli aniqlangan funktsiyani chizing: $f(x) = \begin{cases} x^2, & x \geq 0 \\ -x^2, & x < 0 \end{cases}$
  3. Ro'yxatdagi eng katta elementni topuvchi algoritm yozing.

Murakkab daraja

  1. Sinus va kosinus funktsiyalarini bir grafikda ko'rsating va kesishish nuqtalarini toping.
  2. Kvadrat tenglama yechuvchi dastur yozing (barcha holatlari bilan).
  3. Animatsiyali grafik yarating: harakat qiluvchi nuqta yoki to'lqin.

Loyiha

Matematik simulyatsiya yarating:

  • Monte Carlo usuli bilan π qiymatini hisoblang
  • Tasodifiy yurish (random walk) simulyatsiyasi
  • Oddiy differensial tenglamani raqamli yechish
Yechimlarga yo'l-yo'riq (ko'rish uchun bosing)
