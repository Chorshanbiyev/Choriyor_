Mavzu: rkfixed · odesolve · Solve Block · Numerical solving

MathCAD — bu muhandislik va ilmiy hisob-kitoblarni bajarish, matematik modellar tuzish, formulalarni tahrirlash va ularni grafik tarzda tasvirlash uchun mo‘ljallangan kompyuter algebra tizimi (CAS) hisoblanadi. Uni dastlab PTC (Parametric Technology Corporation) kompaniyasi ishlab chiqqan. Dastur foydalanuvchilarga matematik ifodalarni tabiiy shaklda — ya’ni qog‘ozda yozilgandek ko‘rinishda kiritish imkonini beradi.

Interaktiv modulga o'tish MathCAD ishchi varaqlari

MathCAD'da differensial tenglamalar

Asosiy xususiyatlari: Interaktiv muhit — formulalar, matnlar, grafiklar va jadvallar bitta hujjatda joylashadi, natijalar avtomatik hisoblanadi. Simvolik va sonli hisoblash — algebraik tenglamalarni yechish, integrallar, differensial tenglamalar, matritsalar bilan ishlash imkonini beradi. Grafik tasvirlash — 2D va 3D grafiklar chizish, funksiyalarni tahlil qilish hamda natijalarni vizual shaklda taqdim etish mumkin. O‘zgaruvchan birliklar tizimi — fizik kattaliklarni o‘lchov birliklari bilan birgalikda ishlatish va avtomatik tarzda ularni o‘zgartirish funksiyasi mavjud. Hujjatga yo‘naltirilgan interfeys — barcha hisob-kitoblar, formulalar, izohlar va natijalar bir sahifada tartibli saqlanadi, bu esa hisobot yoki ilmiy ishlar tayyorlashni osonlashtiradi.

MathCAD’ning o‘quv jarayonidagi ahamiyati. Talabalar murakkab matematik ifodalarni aniq ko‘rinishda yozish va ularning grafik ko‘rinishini olish imkoniga ega bo‘ladi. Fizika, mexanika, iqtisodiyot, qurilish, axborot texnologiyalari kabi fanlarda modellashtirish, tahlil va loyihalash ishlarini avtomatlashtiradi. Hisob-kitob natijalari avtomatik yangilanadi, bu esa xatolik ehtimolini kamaytiradi va tahliliy fikrlashni rivojlantiradi.

MathCAD ODEs Numerical Runge-Kutta

MathCAD ishchi varag'i namunalari

1) Birinchi tartibli ODE - rkfixed

first_order_ode.xmcd

Differensial tenglama: dy/dx = -2y + x
Boshlang'ich shart: y(0) = 1
D(x, y) := -2·y + x
x₀ := 0
x₁ := 2
y₀ := 1
N := 100
Yechim:
sol := rkfixed(y₀, x₀, x₁, N, D)
Natija: 101×2 matritsa (x, y qiymatlari)

2) Ikkinchi tartibli ODE

second_order_ode.xmcd

Tenglama: d²y/dx² + 4y = 0
Boshlang'ich shartlar: y(0) = 1, y'(0) = 0
D(x, Y) := [Y₁, -4·Y₀]
x₀ := 0
x₁ := π
Y₀ := [1, 0]ᵀ
N := 200
Yechim:
sol := rkfixed(Y₀, x₀, x₁, N, D)
Y₀ = y(x), Y₁ = y'(x)

3) Solve Block bilan ODE

ode_solve_block.xmcd

Boshlang'ich taxmin:
y(x) := e^(-x)
Given bloki:
Given
d/dx y(x) = -y(x) + sin(x)
y(0) = 1
Yechim:
Y := odesolve(x, 2)
Y(x) - yechim funksiyasi

4) Tenglamalar sistemasi

system_of_odes.xmcd

Sistema:
dx/dt = -0.1·x + 0.2·y
dy/dt = 0.1·x - 0.2·y
D(t, Y) := [-0.1·Y₀ + 0.2·Y₁]
[0.1·Y₀ - 0.2·Y₁]
Y₀ := [10, 5]ᵀ
t₀ := 0, t₁ := 20, N := 200
Yechim:
sol := rkfixed(Y₀, t₀, t₁, N, D)

5) Chegaraviy qiymat masalasi

boundary_value_problem.xmcd

Tenglama: d²y/dx² - y = x
Chegaraviy shartlar: y(0) = 0, y(1) = 1
y(x) := x
Given bloki:
Given
d²/dx² y(x) - y(x) = x
y(0) = 0
y(1) = 1
Yechim:
Y := odesolve(x, 1)

6) Stiff tenglamalar

stiff_equations.xmcd

Stiff sistema:
dy₁/dt = -1000·y₁ + 1000·y₂
dy₂/dt = y₁ - y₂
D(t, Y) := [-1000·Y₀ + 1000·Y₁]
[Y₀ - Y₁]
Y₀ := [1, 0]ᵀ
Stiff yechuvchi:
sol := Stiffr(Y₀, 0, 0.1, 100, D)
Stiffr - qattiq tenglamalar uchun

Mustaqil ishlar va masalalar

Boshlang'ich daraja

  1. $\frac{dy}{dx} = 2y$ tenglamasini rkfixed bilan yeching, $y(0) = 1$.
  2. $\frac{dy}{dx} = -y + \sin(x)$ tenglamasini odesolve (Solve Block) bilan yeching.
  3. $\frac{d^2y}{dx^2} + 4y = 0$ ikkinchi tartibli tenglamasini yeching, $y(0) = 1, y'(0) = 0$.
  4. Logistik o'sish modeli: $\frac{dy}{dx} = 0.1y(1 - \frac{y}{100})$ ni tahlil qiling.

O'rta va yuqori daraja

  1. Predator-Prey sistem modeli: Lotka-Volterra tenglamalarini yeching.
  2. Van der Pol osilatatorini ($\ddot{x} - \mu(1-x^2)\dot{x} + x = 0$) o'rganing.
  3. Chegaraviy qiymat masalasini Solve Block bilan yeching.
  4. Stiff tenglamalar sistemasini Stiffr bilan yeching.
  5. Parametrik tahlil: $k$ qiymatining o'zgarishini o'rganing.
Yechimlar va maslahatlar

1-masala (eksponensial o'sish):

D(x, y) := 2·y
sol := rkfixed(1, 0, 2, 100, D)
